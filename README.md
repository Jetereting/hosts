# hosts
go语言写的win/linux平台下的翻墙软件（通过修改hosts） 
####Windows平台：
双击go.exe即可(自动管理员身份运行 用到了rsrc)
####Linux平台：
chmod 111 main  赋予执行权限<br>
su 切换到root权限<br>
./main 执行<br>
若无 ERROR 则成功<br>
####Mac平台：
sudo ./mac
